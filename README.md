# ApiLunchLite
Un orquestador y gestor de API ligero, desarrollado en Go. Diseñado para gestionar entornos de desarrollo locales sin la sobrecarga de Docker. Incluye detección automatizada del entorno Python, gestión de procesos (seguimiento de PID) y registro localizado. de procesos (seguimiento de PID) y registro localizado."

## 📋 Requisitos
* **Sistema Operativo:** Windows (probado en Windows 10/11).
* **Lenguaje:** Go 1.21+ (solo si vas a compilar desde el código fuente).
* **Dependencias:** [SQLite](https://www.sqlite.org/) (el sistema crea el archivo `.db` automáticamente).

⚠️ Nota: Actualmente, ApiLunchLite está diseñado exclusivamente para entornos Windows.
